Informations to Start

With this plugin you can pricing the files and groups to selling it , and recive the payments to your paypal or Stripe account automaticly ( WebMasters Only )

You need only : PayPal Business Account or Stripe account, and you can create an REST API APP for paypal or get Stripe API keys # for developers : you can create a paypal app if your account is personal , but you will use the sandbox API (TEST) until you upgrade your account to buisness and for Stripe you will use the test api keys until you active your account

After you create your REST API APP you will got ( Client ID ) and ( Client Secret ) . set them in {olang.CONFIG_KLJ_MENUS_KLEEJA_PAYMENT} , and the ( Currency Code ) it have to be 3 Letter Only EX: USD , EUR , CHF check the accepted currencies code here

To pricing a file you neet to go to ( Pricing a File ) in the Side menu , with the file id or file link you can open file details , after clicking file details you will see some details about the file and at the end you will see a form to set the price , and if the file was paid , you can change the price or set the price to 0 (zero) to make the file for free again
and if you want to set a price for a group go to ( users & Groups -> Edit Data ) and you will find ( Join price ) .
Note : the default groups and the default registeration group can not be paid

File Owner and website founders can download the file for free ( Founder Role is not admins group , the founders can give founder Role to any member in kleeja even if he is not in admins group )

After you open kj_payments CP you can find some informations about payments ( Transactions numbers and amounts , and Net Profits )
all transactions : is the number of all approved payments when you started using this plugin ,
monthly transactions : is the number of all payments of this month with the net profits and it will be reseted evry month
daily transactions : is the number of all payments of today only and it will be reseted evryday
you can access to the old payments and amounts for any day or month by archives . (to access to monthly archive , make the day empty and select the month and year )
Only the website founders can access to kj_payments control Panel
Note : if you see that a method display the number of amount without writting ( Net Profit ) that mean the amount is writed with fees
